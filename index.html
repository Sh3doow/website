ARP Defense

This script employs the Scapy library to observe network activity and pinpoint possible instances of ARP Spoofing attacks. ARP Spoofing involves a malicious actor sending counterfeit ARP messages across the network, intending to link their own MAC address with genuine IP addresses. This activity can result in communication interruptions or the interception of data.

Data Integrity

This script acts like a digital detective for downloaded files. It calculates a unique fingerprint, called a hash, for the file you downloaded and compares it to the expected fingerprint. If they match, it's like getting a thumbs-up that the file is authentic. If not, it's a red flag that something might be wrong with the file, like tampering or corruption.

Mac Change

This script facilitates the alteration of MAC addresses for network adapters on Windows systems. It offers options to either generate a random MAC address or specify a custom one. Users can select the target network adapter from a list of connected adapters and then execute the MAC address change. Additionally, it disables and re-enables the adapter to ensure the modifications take effect.

Zip Locker

This program creates or updates a password-protected ZIP file. It accepts the ZIP file name and files to add via command line arguments. Erase Malware

Zip Breaker
